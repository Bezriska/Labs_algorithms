## Теоретическая часть

**Бинарное дерево поиска (Binary Search Tree, BST)** — это структура данных, в которой:
- Каждый узел содержит значение и ссылки на левого и правого потомков
- Для любого узла все значения в левом поддереве меньше значения узла
- Для любого узла все значения в правом поддереве больше значения узла

### Сложность операций

| Операция | Средний случай | Худший случай |
|----------|----------------|---------------|
| Поиск    | O(log n)       | O(n)          |
| Вставка  | O(log n)       | O(n)          |
| Удаление | O(log n)       | O(n)          |

Худший случай возникает при вырожденном дереве (например, когда элементы добавляются в отсортированном порядке).

## Описание реализации

### Структура данных

```c
typedef struct node {
    int data;           // Значение узла
    struct node* left;  // Указатель на левого потомка
    struct node* right; // Указатель на правого потомка
    int depth;          // Глубина узла в дереве
} Node;
```

### Реализованные функции

#### 1. `initiate_tree(int head_data)` 
Создает корень дерева с заданным значением.

**Алгоритм:**
- Выделяет память для нового узла
- Инициализирует поля структуры
- Возвращает указатель на созданный узел

**Сложность:** O(1)

#### 2. `add(int data, Node* head)`
Добавляет новый элемент в дерево.

**Алгоритм:**
- Начинает с корня дерева
- Если значение меньше текущего узла — идет влево
- Если значение больше текущего узла — идет вправо
- Продолжает до нахождения свободной позиции
- Создает новый узел в найденной позиции

**Сложность:** O(log n) в среднем, O(n) в худшем случае

#### 3. `search_data(int data, Node* head)`
Ищет элемент в дереве.

**Алгоритм:**
- Рекурсивно сравнивает искомое значение с текущим узлом
- Если значения равны — элемент найден
- Если искомое меньше — поиск в левом поддереве
- Если искомое больше — поиск в правом поддереве

**Сложность:** O(log n) в среднем, O(n) в худшем случае

#### 4. `delete_node(int data, Node* head)`
Удаляет узел из дерева.

**Алгоритм:**
- Рекурсивно находит узел для удаления
- Удаляет узел, если у него нет потомков
- Освобождает память

**Сложность:** O(log n) в среднем, O(n) в худшем случае

#### 5. `check_depth(Node* head)`
Вычисляет максимальную глубину дерева.

**Алгоритм:**
- Рекурсивно обходит все узлы дерева
- Отслеживает максимальное значение глубины
- Возвращает максимальную глубину

**Сложность:** O(n)

#### 6. `count_width(Node* head)`
Вычисляет максимальную ширину дерева.

**Алгоритм:**
- Находит максимальную глубину дерева
- Для каждого уровня подсчитывает количество узлов
- Возвращает максимальное количество узлов на одном уровне

**Сложность:** O(n²)

#### 7. `print_tree(Node* head)`
Визуализирует дерево в консоли.

**Алгоритм:**
- Рекурсивно обходит дерево (правое поддерево → корень → левое поддерево)
- Использует отступы для визуализации структуры дерева

**Сложность:** O(n)

## Сборка проекта

### Makefile

Проект использует Makefile для автоматизации сборки:

